To enable seamless interaction with dApps using Web3 Wallets, you must first install Cobo Connect, a Chrome extension that functions as the gateway between Cobo Portal and the dApp ecosystem. For more information, see [Interact with dApps through Cobo Connect](/en/cobo-connect/interact-with-dapps).
